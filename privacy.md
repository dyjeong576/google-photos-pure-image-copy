# Pure Image Copy Privacy Policy

**Last updated:** May 26, 2026

Pure Image Copy is a Chrome extension designed to help users copy images from supported websites as plain PNG image data. This Privacy Policy explains what user data the extension handles, how that data is used, and whether it is shared.

---

## Summary

| Topic | Policy |
| --- | --- |
| Personal information | Not collected |
| Analytics or tracking | Not used |
| Advertising | Not used |
| Remote code | Not used |
| Data sale | Never sold |
| Data sharing | Not shared with third parties |
| Server transmission | No user data is sent to a developer-operated server |

---

## User Data Collected

Pure Image Copy does **not** collect personal information, including:

- Names
- Email addresses
- Account identifiers
- Financial information
- Health information
- Authentication information
- Personal communications
- Precise location

The extension does not collect browsing history for analytics or tracking. It does not use advertising trackers, analytics services, or third-party tracking services.

---

## User Data Processed

When a user intentionally clicks the extension button or selects the context menu action, Pure Image Copy may access:

- The active tab
- The selected or relevant image
- The image URL or page content needed to complete the requested copy action

This access is used only to find the image selected by the user, fetch or convert the image when necessary, and write PNG image data to the user's clipboard.

Pure Image Copy does not use this data to identify users, build user profiles, monitor browsing activity, or serve advertising.

---

## How User Data Is Used

User data handled by the extension is used only for the extension's single purpose:

> Copying an image chosen by the user as plain PNG image data.

Image data is processed for the requested copy operation and is not retained by the extension after the operation is complete.

---

## Stored Data

Pure Image Copy may store user preferences and site access settings using Chrome's extension storage.

This stored data is used only to:

- Remember extension settings between browser sessions
- Apply site access choices selected by the user
- Keep user-selected options available after the browser is restarted

Stored settings remain in the user's browser environment unless removed by the user through the extension options, browser settings, or extension removal.

---

## Clipboard Access

Pure Image Copy writes image data to the clipboard only after the user requests a copy action.

The extension does not read clipboard contents for analytics, tracking, advertising, profiling, or unrelated purposes.

---

## Remote Code

Pure Image Copy does not use remote code.

All JavaScript required by the extension is included in the extension package submitted to the Chrome Web Store. The extension does not:

- Load external scripts
- Import remote modules
- Use WebAssembly
- Evaluate remote or dynamic code with `eval()` or similar mechanisms

---

## Data Sharing

Pure Image Copy does not sell, rent, transfer, or share user data with:

- The developer
- Advertisers
- Analytics providers
- Data brokers
- Other third parties

No user data is transmitted to an external server operated by the developer.

Data may be handled by Chrome and the user's browser environment as part of normal extension operation, including Chrome extension storage and clipboard APIs.

---

## Limited Use

The use of information received from Chrome APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

---

## Contact

If you have questions about this Privacy Policy, please contact the developer through the Chrome Web Store listing for Pure Image Copy.
