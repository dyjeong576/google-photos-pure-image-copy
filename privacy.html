Pure Image Copy Privacy Policy

Last updated: May 26, 2026

Pure Image Copy is a Chrome extension designed to help users copy images from supported websites as plain PNG image data. This Privacy Policy explains what user data the extension handles, how that data is used, and whether it is shared.

User Data Collected

Pure Image Copy does not collect personal information such as names, email addresses, account identifiers, financial information, health information, authentication information, personal communications, or precise location.

The extension does not collect browsing history for analytics or tracking. It does not use advertising trackers, analytics services, or third-party tracking services.

User Data Processed

When a user intentionally clicks the extension button or selects the context menu action, Pure Image Copy may access the active tab, the selected or relevant image, and the image URL or page content needed to complete the requested copy action.

This access is used only to find the image selected by the user, fetch or convert the image when necessary, and write PNG image data to the user's clipboard. The extension does not use this data to identify users, build user profiles, monitor browsing activity, or serve advertising.

How User Data Is Used

User data handled by the extension is used only for the extension's single purpose: copying an image chosen by the user as plain PNG image data. Image data is processed for the copy operation and is not retained by the extension after the operation is complete.

Stored Data

Pure Image Copy may store user preferences and site access settings using Chrome's extension storage. This data is used only to remember extension settings between browser sessions and to apply site access choices selected by the user.

Stored settings remain in the user's browser environment unless removed by the user through the extension options, browser settings, or extension removal.

Clipboard Access

The extension writes image data to the clipboard only after the user requests a copy action. It does not read clipboard contents for analytics, tracking, or unrelated purposes.

Remote Code

Pure Image Copy does not use remote code. All JavaScript required by the extension is included in the extension package. The extension does not load external scripts, import remote modules, use WebAssembly, or evaluate remote code.

Data Sharing

Pure Image Copy does not sell, rent, transfer, or share user data with the developer, advertisers, analytics providers, data brokers, or other third parties.

No user data is transmitted to an external server operated by the developer. Data may be handled by Chrome and the user's browser environment as part of normal extension operation, including Chrome extension storage and clipboard APIs.

Limited Use

The use of information received from Chrome APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Contact

If you have questions about this Privacy Policy, please contact the developer through the Chrome Web Store listing for Pure Image Copy.
